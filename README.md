## Project Overview
This project demonstrates Java OOP fundamentals including encapsulation,
constructors, methods, arrays of objects, and class composition. The system
models students and courses while performing GPA and credit calculations.

## Class Descriptions
### Student
Stores student information such as name, ID, major, GPA, and credits.
Encapsulation is enforced through private fields and public getters/setters.

### Course
Represents a course that contains an array of Student objects, demonstrating
composition. Includes methods for GPA averages and credit analysis.
